### Steps to install the Fingerprint App

1. Run "mvn clean install" from the Samples folder for the first time
2. Then cd to the Samples/Biometrics/Android/multibiometric-sample folder
3. Run "mvn clean compile package android:deploy" after connecting the Android device to the machine
4. This should install the App in the device
5. The apk for the app can be found under the Samples/Biometrics/Android/multibiometric-sample/target folder under the name for distrubution
